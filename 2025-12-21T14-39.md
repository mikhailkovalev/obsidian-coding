# Getting started with Vim: the basics
Create or open a file
```bash
vim /path/to/file
```

## Modes
 - Normal - default; for navigation and simple editing
 - Insert - for explicitly inserting and modifying text
 - Command Line - for operations like saving, exiting, etc.
 - ...

### Normal mode
`Esc` switches to Normal mode. Even though you are already in Normal mode
 - `p` - paste copied text
 - `u` - undo last action(?)
 - `dd` - delete a line
 - `Ctrl`+`ww` - switch between panes
 - `$` - jump to the last character in line
 - `o` - insert a new line below current line and switch to Insert mode
 - `v` - highlight the text
 - `y` - copy highlighted text to clipboard

### Command Line mode
Press `:` in Normal mode
 - `q!`quit without saving
 - `x!`  - quit with saving
 - `set number` - show up line numbers
 - `<LINE_NUMBER>` (for example `:2`) - jump to line
 - `$` - jump to the end of file
 - `/<SEARCH_KEYWORD>` - search (use `n` button to get next match)
 - `split <FILE_NAME>` - opens `<FILE_NAME>` in horizontally splitted pane
 - `vsplit <FILE_NAME>` - opens `<FILE_NAME>` in vertically splitted pane

### Insert mode
`i` - switch to Insert mode (Vim shows `-- INSERT --` in the lower-left)
