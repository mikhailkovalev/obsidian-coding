#iptables #habr 
[Source](https://habr.com/ru/articles/747616/)
iptables имеет 4 основные таблицы:
 - filter - для фильтрации пакетов
 - nat - для настройки NAT (Network Address Translation)
 - mangle - для спец.обработки пакетов
 - raw - для обхода системы отслеживания состояний

Каждая таблица состоит из набора цепочек. Цепочки -- это последовательности правил, которые применяются к пакетам. Есть три встроенные цепочки
 - INPUT -  для пакетов которые предназначены для самой системы
 - FORWARD - для пакетов, которые проходят через систему
 - OUTPUT - для пакетов, которые исходят из системы

Чтобы посмотреть все текущие правила 
```bash
iptables -L
```
