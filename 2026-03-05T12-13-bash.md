### Переменные среды
Можно использовать в двойных кавычках, это не помешает системе распознать их
```bash
echo "Home: $HOME"
```
Экранировать символ $: `\$`

### Переменные среды
Хранят значение пока не завершится скрипт
```bash
grade=5
person="Adam"
echo "$person has $grade"
```

### Подстановка команд
```bash
mydir=`pwd`
echo $mydir
```
```bash
mydir=$(pwd)
echo $mydir
```

### Математические операции
```bash
var1=$(( 5 + 5 ))
echo $var1

var2=$(( $var1 / 3 ))
echo $var2
```

### Условный оператор
```
if команда1
then
команды
elif команда2
then
команды
else
команды
fi
```
Пример 1:
```bash
if pwd
then
echo "it works"
fi
```
Пример 2:
```bash
user=kovalev_m
if grep $user /etc/passwd
    then
      echo "user $user exists"
    else
      echo "user $user does not exist"
fi
```
Пример 3:
```bash
#!/bin/bash
user=kovalev_m
if grep $user /etc/passwd; then
  echo "user $user exists"
elif ls /home; then
  echo "user $user does not exist but his directory does"
else
  echo "user $user does not exist"
fi
```
