#linux #cli #mouse #gpm #copy #paste
## Install `gpm` (if not already installed)
```bash
sudo apt install gpm
```
## Use the mouse
- **Select (Copy):** Press the left mouse button and drag to highlight the text you want to copy. The text is automatically copied to `gpm`'s buffer upon selection
- **Paste:** Press the middle mouse button (scroll wheel) or right mouse button if you have a two-button mouse

### Alternative without `gpm` (using `screen` or `tmux`)
- Start a new screen session by running `screen`
- Enter copy mode by pressing `Ctrl`+`A`, then `Esc`
- Move the cursor to the start of the text you want to copy and press `Space` to begin selection
- Move the cursor to the end of the text and press `Space` again to finish copying
- Exit copy mode and paste the text by pressing `Ctrl`+`A`, then `]`
