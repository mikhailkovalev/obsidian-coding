#python #mock #spy #method

Spy instance method call:
```python
from unittest import mock

with mock.patch.object(
    my_obj,
    'my_method',
    wraps=my_obj.my_method,
) as my_method_spy:
    ...
```


```python
with mock.patch.object(
    MyClass,
    'my_method',
    side_effect=MyClass.my_method,
    autospec=True,
) as fake:
    ...
```
